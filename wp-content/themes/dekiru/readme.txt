== Description ==
dekiru is a simple WordPress theme bulit by Bootstrap. It supports tablets and smartphones in the responsive Web design. By theme customize, setting slider you can easily.


== Features ==
Bootstrap based,Supports customizer, Responsive Layout, Page Navigation, Breadcrumb Navigation, and more!


=== Widgets Areas ===
The Theme has 6 widget areas.
You can use these area to customize the content of your website.
